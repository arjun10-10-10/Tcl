# My Tcl Programming Journey 🚀

Welcome to my repository! This project tracks my daily coding practice as I learn Tcl (Tool Command Language) from scratch using Exercism.

## 🎯 Goal
My target is to spend **1 hour per day for 30 days** mastering Tcl syntax, data structures, and string parsing to build a strong foundation for VLSI physical design and EDA tool scripting.

## 📁 Repository Structure
The code is organized by daily topics and individual Exercism exercises:
* `/Hello-World` - Initial configuration and basic output syntax.
* `/Strings-Regex` - Regular expressions and log-parsing practice.
* `/Lists-Arrays` - Deep dive into handling complex data structures and dictionaries.

## 🛠️ Tools Used
* [Exercism Platform](https://exercism.org/) - Online interactive coding environment and mentor feedback.
* Markdown for documentation.
